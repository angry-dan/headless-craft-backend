This is a [Craft CMS](https://craftcms.com/) project.

## Getting Started

Run:

```bash
composer install
```

To get the site going you'll need Docker running. Then:

```bash
docker-compose up
```

Open [http://localhost:1000](http://localhost:1000) with your browser to see the result.

## What is this thing then?

An experiment in running a site on a headless CraftCMS backend and a React frontend.
Backend here: https://github.com/angry-dan/headless-craft-backend
Frontend here: https://github.com/angry-dan/headless-craft-frontend

Concepts/tools:

- Craft CMS
- Craft in headless mode
- Craft's GraphQL API
- Apollo client
- Next.js
- Apollo generating Typescript types
- React
